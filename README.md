* Textbox (abstrakt klasse / interface)

  * Redigerbar
* Meny 
* TextGetter: Henter text sender til Textbox og TextScrambler 
* TextScrambler: Endrer på teksten og sender til Textbox

* Vim: får input, oppdaterer Redigerbar tekst
